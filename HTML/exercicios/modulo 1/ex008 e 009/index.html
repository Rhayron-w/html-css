Principais formatações em textos:

Textos em negrito ou em destaque

Nessa frase é utilizado negrito a partir do uso da tag <b> e </b>. Ela não é uma tag semântica.

Nessa frase agora é utilizado destaque ou negrito a partir do uso da tag <strong> e </strong>. Essa é uma tag semântica para usar em HTML5.

Textos em itálico ou ênfase

Nessa frase estou utlizando o texto em itálico com a tag <i> e </i>. Essa é a forma não semântica dele.

Nessa frase agora foi utilizado o texto em ênfase ou itálico usando a tag <em> e </em>. Essa é a forma semântica.

Textos com marcação

Nessa frase foi utilizado a marcação de textos utlizando a tag <mark> e </mark>.

Para mudar a cor da marcação, é usado CSS com o parâmetro style dentro da tag <mark> e com o tipo background-color.

A marcação do texto agora está com uma cor diferente graças ao CSS.

Para deixar todos textos marcados, usa a tag style na parte de head do html junto com o mark e background-color para deixar todos com essa cor

Textos maiores e menores

Algumas tags podem aumentar ou diminuir o tamanho dos textos. Apesar de que a tag <big> está em desuso, enquanto a tag <small> ainda é comumente usada.

Texto deletado

Antigamente era utilizado a tag strike para deixar o texto em formato deletado. A forma <strike> e </strike> não é semântica e é pouco usada.

Já a tag del como nova pode ser usada normalmente usando a tag <del> e </del>. Essa forma é semântica.

Texto inserido ou sublinhado

Usando a tag antiga u, o texto fica inserido, mas a forma <u> e </u> não é semântica.

Já a tag nova ins, deixa o texto também sublinhado, mas de forma semântica com <ins> e </ins>.

Textos subescritos e supescritos

Para fazer textos químicos e matemáticos como x2 é utilizado a tag <sup> e </sup>.

Para fazer textos como H2O em química, é utilizado a tag <sub> e </sub>.

Exemplos para códigos

Para ajudar programadores em ler mais facilmente os códigos de outro programa, é utilizado a tag <code> e </code>.

O texto vai ficar mais ou menos assim:
int main()
{
int num;
printf("Escolha um número: ");
scanf("%d",&num);
if(num % 2 == 0)
{
printf("seu número é par.\n");
}
else
{
printf("seu número é impar.\n");
}
return 0;
}
