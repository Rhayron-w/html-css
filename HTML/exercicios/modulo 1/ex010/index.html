Aprendendo listas

Listas ordenadas

A ordem faz diferença nessa lista.

Para criar uma lista ordenada, utiliza a tag <ol> e </ol>.
E dentro de ol, para cada item dessa lista, você utiliza a tag <li> e </li>.

EXEMPLO:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Se arrumar
  5. Ir para o trabalho

Listas não ordenadas

Para criar uma lista não ordenada, utiliza a tag <ul> e </ul>. Dentro de ul, também se utiliza a tag <li> e </li>.

EXEMPLO:

  • Pão
  • Alface
  • Tomate
  • Queijo
  • Maionese
  • Ketchup
  • Carne de hamburguer

Mistura de listas 1.0

Pode haver mistura de listas para ajudar em suas criações.

EXEMPLO:

  1. Programação
    • HTML
    • Python
    • C
  2. Design
    • CSS
    • Photoshop
    • Illustrator
